# Ejecutar en desarrollo

1. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

2. Clonar el repositorio
```
git clone https://github.com/VazquezAriel/consultas-medicas.git
```

3. Ejecutar
```
npm install
```

4. Levantar la base de datos
```
docker-compose up -d
```

5. Ejecutar en modo desarrollo
```
npm run start:dev
```
